# artlu's Nano Farchive web site

This is a simple Astro Starlight SSG built around Obsidian markdown files.

h/t [starlight-obsidian](https://github.com/HiDeoo/starlight-obsidian) plugin by [HiDeoo](https://github.com/HiDeoo) and [starlight-site-graph](https://github.com/fevol/starlight-site-graph) plugin by [Fevol](https://github.com/fevol)!

## RUNNING

- create a Farchive or a Nano Farchive

- open in Obsidian to generate Obsidian dotfiles

- place obsidian folder in `./src/contents/obsidian`

- run two one-off scripts to fix old imgur links and re-point relative links

- create and deploy Astro Starlight site
